## Usage
Clone this repository.

Enter directory of project then install depedency.
```shell
cd one-line-layouts
```

then
```shell
npm install
```

Run local development
```shell
npm run serve
```

Build for `development` env
```shell
npm run dev
```

Build for `production` env
```shell
npm run build
```
